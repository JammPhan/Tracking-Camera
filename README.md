# Tracking-Camera
Documenting the Development of a Tracking Camera.

Parts used:

- 2 x MG996R servos
- PCA9685 Servo driver board
- Raspberry Pi 4 (4GB)
- Pi camera module v3
  
Week 1:

Goals: Learn CAD and assemble hardware. Try understand all the components of the hardware.

Progress:

Learnt some CAD, using OnShape software, allowing me to make shapes and screw holes to make the pan-tilt axis frame to connect the servos and cameras to. Design inspired by common ones found online.

Printed the parts and connected the hardware together. Checking that everything is working by taking a photo with the camera and testing the servo motion by moving them to different angles.

What I have learnt:

Improvements: Add a space to put Raspberry Pi board into tracking camera and servo driver, add some tubes to clean up wires, Learn more features of CAD to look for more improvements to the frame design.



Week 2:

Goals: Learn some OpenCV and achieve tracking of a tennis ball (coloured tracking).
